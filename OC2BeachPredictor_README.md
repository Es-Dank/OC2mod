# OC2 Beach Menu Predictor

**OC2 Beach Menu Predictor** is a small BepInEx-based utility mod for *Overcooked! 2*. It was designed for the Beach Special / Beach 3-4 style level and provides a compact in-game menu preview panel.

The mod was created as an experimental quality-of-life tool for private testing and casual play. It focuses on one specific use case: showing a rolling list of upcoming beach-level orders so players can prepare ingredients before the order appears on screen.

## Features

- Predicts and displays the upcoming menu sequence for the supported beach level.
- Keeps a rolling preview list, so newly generated orders replace the ones that have already appeared.
- Uses a compact in-game overlay designed to avoid blocking the main play area.
- Includes a force/lock mode for private testing, where the displayed order queue is intended to match the generated order queue.
- Designed for BepInEx-based *Overcooked! 2* modding environments.

## Intended Use

This mod is intended for:

- private lobbies;
- casual practice;
- testing order-generation behaviour;
- experimenting with *Overcooked! 2* modding mechanics.

It is not intended for public matchmaking, competitive scoring, leaderboard submission, or any situation where other players have not agreed to use gameplay-altering mods.

## Compatibility Notes

This project was developed around the Beach Special / Beach 3-4 style level. It may not work correctly on other levels, custom maps, or heavily modified arcade environments.

Other BepInEx mods that modify order generation, recipe weights, level loading, or arcade lobby behaviour may conflict with this mod. If the displayed orders do not match the actual orders, test the mod in a clean BepInEx setup first.
